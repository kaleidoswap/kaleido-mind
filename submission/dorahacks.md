## Short description

KaleidoMind is an Apache-2.0 local-first reasoning and tool-calling engine for
Bitcoin wallets. It uses QVAC for local LLM, embedding, speech-to-text and
text-to-speech inference. Deterministic recipes keep small models reliable,
while spend-capable tools always pause for host-enforced confirmation. The same
engine powers the desktop product and the Rate React Native wallet.

## Prior work disclosure

KaleidoSwap, Rate and parts of the wallet/tooling stack existed before the
hackathon. During the hackathon the team built and integrated the shared
KaleidoMind engine, QVAC inference/delegation, tiered funnel, recipes, skill
scoping, safety gates, evaluation harness and cross-surface product experience.
Repository history remains public so judges can audit this boundary.
