# Demo video script (maximum 5 minutes)

- **0:00–0:35 — Problem and architecture:** small local models, deterministic recipes, shared tool contract and structural confirmation.
- **0:35–2:15 — Desktop:** load QVAC model, show hardware/status, run a portfolio or trade request, inspect tool selection and confirmation.
- **2:15–3:30 — Rate on iPhone 17:** show local model status, continue or delegate the workflow, execute wallet tools on-device, export evidence.
- **3:30–4:20 — Receipts:** open JSONL telemetry and benchmark manifest; show model lifecycle, prompt, token counts, TTFT, TPS and backend.
- **4:20–4:50 — Reproduction and product:** clean-clone commands, Apache-2.0 engine, public Rate product, privacy/security limits.

Record one continuous standard run where practical. Keep the phone and desktop
clocks visible, avoid seed phrases/addresses, and upload as an unlisted YouTube
video.
